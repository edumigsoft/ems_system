# Auth Feature

Pacote de autenticação e autorização do sistema.

## Componentes

- **auth_shared**: Modelos, contratos e validators compartilhados.
- **auth_server**: Implementação do backend (Rotas, Services, Repositories).
- **auth_client**: Cliente HTTP para comunicação com o backend.
- **auth_ui**: Interface de usuário (Pages, ViewModels).

## Funcionalidades

- Login (Email/Senha)
- Registro de Usuário
- Refresh Token (com Rotação)
- RBAC (Role Based Access Control)
- Permissões por Recurso
