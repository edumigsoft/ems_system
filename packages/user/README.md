# User Feature

Pacote de gestão de usuários do sistema.

## Componentes

- **user_shared**: Modelos, contratos e validators compartilhados.
- **user_server**: Implementação do backend (Rotas, Services, Repositories).
- **user_client**: Cliente HTTP para comunicação com o backend.
- **user_ui**: Interface de usuário (Pages, ViewModels).

## Funcionalidades

- CRUD de Usuários
- Perfil de Usuário
- Gestão de Permissões
