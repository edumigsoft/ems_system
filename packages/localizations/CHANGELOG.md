# Changelog

Este arquivo documenta as mudanças no **Localizations Feature** e seus subpacotes.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/lang/pt-BR/).

---

## [1.0.0] - 2026-01-19

### Added
- Estrutura inicial do Localizations Feature
- Subpacotes: localizations_shared, localizations_server, localizations_ui
- Documentação completa com estratégia multi-sistema
- Roadmap de internacionalização

### Planned
- Suporte a múltiplos idiomas (pt_BR, en_US, es_ES, fr_FR)
- LocalizationDelegate para Flutter
- Formatadores de data, número e moeda
- Backend para gestão de traduções
- Interface administrativa para edição
- Export/Import de traduções
- Suporte a RTL (Right-to-Left)
- Cache de traduções
